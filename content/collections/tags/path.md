title: Path
overview: Prepend a URL/string with the current site root.
parameters:
  -
    name: src
    type: string
    description: |
      The path to append to the site root.

## Usage {#usage}

```
<a href="{{ path src='contact' }}">Contact us</a>
```

```
<a href="http://example.com/contact">Contact us</a>
```
